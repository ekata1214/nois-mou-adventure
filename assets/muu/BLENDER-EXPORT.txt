## 体型が女性デフォルトになる（HumGen Process が効かない）

viewport ではムー君なのに GLB だけ `Female_Body_Default` になる場合:

**原因:** HumGen の体型は Shape Keys / Live Keys。GLB はスライダーを export しない。
ARP リマップ後、HumGen **Apply Modifiers** が `drivers` エラーで落ちることが多い。

### 対処 A — viewport 体型を焼く（推奨）

1. **blend を別名保存**（戻せるように）
2. タイムラインを **フレーム 1（Rest/T-pose）** に
3. ターミナル:

```bash
cd "/Users/takaearasaki/Desktop/MY GAMES/nois-mou-adventure"
BLENDER_BIN="/Applications/Blender.app/Contents/MacOS/Blender"
"$BLENDER_BIN" assets/muu/speak-mou.blend --python scripts/blender/bake-muu-body.py
```

4. Blender が開いたら viewport で体型確認 → **Ctrl+S**
5. GLB export → `speak-mou6.glb`

（Blender 内 Scripting タブから `scripts/blender/bake-muu-body.py` を Run でも可）

### 対処 B — Shape Keys を手動 Apply

体メッシュ選択 → Object Data → Shape Keys → **Apply All Shape Keys**
（スライダーに値が入っている場合のみ有効）

### HumGen Process の使い分け

| 処理 | 用途 |
|------|------|
| Apply Modifiers | ARP 後は **エラーになりやすい** → 使わない |
| Bake Textures | テクスチャ用（体型には効かない） |

---

## テクスチャ（HumGen Bake Textures）

1. HumGen → Process → **Bake Textures** のみ
2. Pack Resources
3. Brain: `brain-baked` → Principled Base Color（別途 Diffuse Bake）

---

## Brain（脳）

1. Shading → `brain-baked` Image Texture ノードを **選択**
2. Render Properties → Bake → **Emit**（Diffuse が黒い場合）
3. Color → Principled Base Color → Pack → export

---
