# 殻ライフ / 採集・クラフト（構想）

> 戦闘・アクションとは**別ピラー**。  
> 「女性ユーザーも楽しめる」= 狭い意味の性別ターゲットではなく、**競争・暴力以外の遊び方**としての生活・育成・表現。

## ゼルダから踏襲するもの

- 武器（NOU）だけが強くなるのではなく、**拠点（殻）が育つ**
- 採集 → クラフト → 空間が変わる → ムー君の生活が変わる
- 「別荘」「庭」= 本拠を超えた**第二の自分の場所**

## 殻の中でやりたいこと（リスト）

| カテゴリ | 内容 |
|---------|------|
| **移動** | タップした場所にムー君が寄る（3D/2D 殻内ポイント移動） |
| **メモ** | 壁にメモを貼る・読み返す（プレイヤーの言葉が空間に残る） |
| **ペット** | ムー君が殻の中でペットを飼う |
| **庭** | 野菜を育てる、収穫 |
| **拡張** | 殻のレイアウト・家具・小物 |
| **別荘** | 殻とは別の第二拠点を作れる |

## NOU 側（採集）

- フィールドで素材採集（喜怒哀楽エリアで異なる資源）
- 殻/庭/別荘のクラフト素材
- 戦闘必須にしない — **歩く・触る・拾う** だけでも進む

## 既存コンセプトとの接続

- **殻 = 内省モード** → 生活シムは内省の「実体化」
- **100字回答** → メモ・日記の壁と同系統
- **人間メーター** → 採集・育成・世話も「アクション」として上昇候補
- **闇の思念体** → 放置した庭・ペット・殻が荒れる（優しい警告）

## 実装フェーズ案

### Life 0 — 殻タップ移動
- 殻 3D/2D 上の床をタップ → ムー君が歩いて寄る
- 技術: shell-room 床 raycast + Muu 位置補間

### Life 1 — メモの壁
- プレイヤーが短文/長文を「貼る」→ 壁テクスチャ or 3D plane
- 過去メモの読み返し（日記の壁）

### Life 2 — 採集 + 簡易クラフト
- NOU で pickup → 殻に持ち帰り
- レシピ少数（種・鉢・ランプ）

### Life 3 — 庭
- 殻の外 or 別シーンに菜園グリッド
- 時間経過で成長（リアル時間 or プレイ時間）

### Life 4 — ペット
- クラフト or 遭遇報酬でペット卵
- 殻内で追従・餌・名前

### Life 5 — 別荘
- 人間% or クリア後解放
- 殻と別セーブ領域、レイアウト自由度高め

## UI トーン

- NOU: 緊張・選択・祝福/警告
- 殻ライフ: ゆっくり・触れる・**見せびらかさない**（UNGR の反動としての温かさ）
