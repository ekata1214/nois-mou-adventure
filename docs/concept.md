# ムー君の冒険 — コンセプト（確定版 2026-07-05）

## プレイヤー

**人間であるプレイヤー**が、言語化できない思考の隙間・気づいていない部分を、**ムー君**という媒体を通して自己認識を深める。

- プレイヤー = 外側の人間（身体・時間・社会を持つ）
- NOU = その人間の思考空間
- ムー君 = 未整理の自己（言語化前の塊）

## 表のゴール / 裏のゴール

| | 内容 |
|---|---|
| **表** | ムー君を「人間」にする（人間メーター 100% → 鏡のポータル） |
| **裏** | クリアの有無に関わらず、深く思考・自己分析できたら作者として成功 |

## 人間メーター

- **祝福でもあり警告でもある** — 人間に近づくほど、内面（ムー君）から遠ざかる緊張
- **あらゆるアクション**で上昇（速度はランダムにばらつく）
  - 四択（殺す / 食べる / 無視 / 友達）はすべて人間らしい → どれも同等にカウント
  - 殻での100文字回答、遭遇、アクション戦闘もアクション
- **アクションしない = ゲームをしない** → ゆっくり下降
- 100% で **鏡へ通じるポータル**が開く

## 闇の思念体

**虚無・孤独・盲目**のメタファ。

増える条件:

- アクションを起こさない
- ネガティブな選択（無視、逃げる 等）
- **殻にこもりすぎ** / **NOUに出すぎ**（モード偏り）

## 殻（イントロバート）

意識のモード。内省し自分を見つめる。

- ムー君の質問に100文字以上で答える
- **予定**: 曲を聴かせ合う、絵を見せ合う、映画のあらすじを紹介し合う
- **予定**: AI会話でムー君と対話

## NOU（エクストロバート）

感情の地形。遭遇と選択。

### アクション戦闘（怒・哀）

現状: 斬るのみ  
**予定**: 話しかける / 撫でる / 見つめる / 守る — 斬ると同列の「人間らしいアクション」

## エンディング — 鏡

1. 人間 100% → 鏡ポータル
2. 最終対峙 = **自分**。アクション + RPG 両方
3. 鏡戦: 勝っても負けても食べても無視しても → **リトライ画面**

### リトライ「はい」

- 鏡のムー君と再戦 → **どんな行動でも勝てる**
- クリア台詞: 「君は自分の選択に責任を持って僕に人間ってものを教えてくれたね。」
- **クリア後**: 新ワールド + プレイヤー同士が森のように互いの殻を行き来

### リトライ「いいえ」

- 死んだムー君が起き上がる
- 「僕は人間になろうとしてるのに、外側にいる君は僕たちのいままでの選択を無下にするんだ」
- **ハードモード**で最初から
- ハードモード: アクションを増やすほど難易度低下 → ムー君再戦可能（必ず勝てる）

## 削除

- シェル質問「気づいてた人いる？」 — 削除済み

## ロードマップ（優先順）

### Phase 1 — NOU フィールド / 戦闘基盤
- 全体的なフィールド修正（地形・背景・読みやすさ）
- 敵の挙動（フィールド上で動く / アクションシーン内の動き）
- アクション戦闘: 斬る以外（話す / 撫でる / 見つめる / 守る）
- RPG 遭遇をもっと「ゲームっぽく」（ターン感・演出・フィードバック）

### Phase 2 — 人間レベル連動
- **人間メーターが上がるほど RPG の選択肢が増える**（低: 2択 → 高: 4択+特殊）
- **アクション戦闘**: 人間レベルに応じて **体力（HP）・攻撃力** が上昇
- 条件達成で **フィールド領域・経路を開放**（下記「開放条件」参照）

### Phase 3 — 殻の拡張
- 曲・絵・映画あらすじ・AI会話（下記アイデア）
- 殻での体験が NOU の能力・開放と双方向にリンク

### Phase 4 — 鏡 / クリア後
- 鏡ボス、リトライ分岐、ハードモード
- クリア後: 森のように互いの殻を行き来

## 人間レベル → 能力スケール（案）

人間メーターを **Tier** に分けて UI も変える:

| Tier | 人間% | RPG | アクション | フィールド |
|------|-------|-----|-----------|-----------|
| 0 静寂 | 0–24 | 基本4択のみ | 斬るのみ、弱い | 交差点 + 1領域 |
| 1 気配 | 25–49 | +「逃げる」 | + 見つめる | + 橋1本 |
| 2 近い | 50–74 | +「話す」系 | + 話す / 撫でる | + 領域1、VOID縁 |
| 3 人間 | 75–99 | + コンボ選択 | + 守る、攻撃↑ | 全領域（鏡除く） |
| 4 鏡 | 100 | 鏡専用 | 全アクション最大 | 鏡ポータル |

- **アクション**: `attackPower = base + human% * 0.12`, `combatHpBonus = human% * 0.08`
- **RPG**: Tier 未満の選択肢はグレーアウトではなく **「まだ言葉がない」** 演出

## フィールド開放条件（案）

「人間%だけ」だと単調なので **殻 + NOU の実績** を混ぜる:

| 解放 | 条件例 |
|------|--------|
| 喜の領域 | 殻で3問回答 |
| 怒の領域 | RPG で1回 encounter |
| 哀の領域 | アクション戦闘1回クリア |
| 楽の領域 | 人間 40% 以上 |
| VOID 縁 | 闇の思念体 60% 超を **殻で1問** して鎮める |
| 鏡 | 人間 100% |

開放時: 交差点に **新しい橋 / 扉** が出現 + Muu の一言。

## 殻モード拡張アイデア

### A. 共有棚（メディア）
- **曲**: URL or ファイル名 → Muu が「重い/軽い/ノイズっぽい」と反応（キーワード or 将来AI）
- **絵**: 画像ドロップ → Muu が色・構図について短い問いを返す
- **映画**: 3行あらすじ → Muu が「主人公は誰に似てる？」と聞き返す  
→ 全部 **100字回答と同列のアクション**（人間メーター上昇）

### B. 日記の壁
- 過去の回答が殻の壁に **文字として蓄積**（読み返せる）
- 一定数で「過去の自分」が質問を **引用して** 返してくる

### C. 呼吸 / 待機
- 何もしないで殻にいる → 闇は増えるが、Muu が rare に独り言
- **意図的な沈黙**も選択肢（内省の一種）

### D. ルームカスタム
- 回答の `kind`（negative/philosophical…）で部屋の小物が増える
- 人間が上がるほど部屋が **雑然 → 生活感** へ（祝福/警告の可視化）

### E. AI Muu（将来）
- 100字回答の **要約を Muu が返す** → プレイヤーが「違う、こういうこと」と訂正
- 訂正もアクション。自己認識の **往復** がゲームになる

### F. 殻から NOU への「持ち出し」
- 殻で聴いた曲名 → NOU フィールド BGM が一時変わる
- 殻で見た絵のモチーフ → 次の encounter アイコンに反映

## RPG をゲームっぽく（方向性）

- ターン開始: 敵が **1歩近づく / セリフ**
- 選択後: **ダメージ数字・画面シェイク・Muu リアクション**
- 人間 Tier で **追加コマンド**（例: 「名前を聞く」「距離を取る」）
- 同じ敵でも **選択履歴** でセリフが変わる（記憶している感）

## アクション戦闘（方向性）

- 敵が **追尾・円形移動・突進**
- 斬る / 話す / 撫でる / 見つめる / 守る — 全部同じ HP バーに効くが **効果と Muu 台詞が違う**
- 人間%で **スタミナ（連続アクション）** と **攻撃力** 上昇
